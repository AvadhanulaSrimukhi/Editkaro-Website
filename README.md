# Editkaro.in Website

## Description
A responsive website for Editkaro.in, a social media marketing and video editing agency.

## Features
- Responsive Home Page
- Portfolio with Video Categories
- About Us Page
- Team Page
- Contact Page
- Email Subscription Form
- Contact Form
- Google Sheets Integration

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Google Apps Script
- Google Sheets

## Deployment
Hosted using Netlify.
